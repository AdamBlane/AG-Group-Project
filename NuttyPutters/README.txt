Control scheme as follows, based on current screen:

Main menu:
Up and down arrows to navitage options.
Space to select option.

Start Game > this screen just has a white triangle over a blue bg
Press 1 to launch game
Press B to go back

Load game > no visible buttons on this screen; view console window
Use left and right to cycle through saved seeds
Enter to select current and load level


Game > 
1 and 2 cycle through cameras (free and chase)
FREE - WASD, QE, SHIFT
CHASE - WASD, Space (hold to build power, release to move ball)
P - Pauses game, while in pause:
  S - Saves game seed
  U - Unpauses
